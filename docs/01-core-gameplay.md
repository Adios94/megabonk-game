# MegaBonk Three.js — 核心玩法设计

## 游戏类型

3D Roguelike Survivor — 第三人称固定视角，自动攻击，移动端优先

## 核心循环

```
选择角色/难度 → 进入竞技场 → 击杀敌人获取经验
      ↑                              ↓
      │                      升级选择强化（武器/典籍）
      │                              ↓
      │                      敌人波次递增 + Final Swarm
      │                              ↓
      │                   找到传送门 → 激活 → 召唤Boss
      │                              ↓
      └──── 结算(银币/任务进度) ←── 通关/死亡
```

## 操作系统

### 移动端（主要）
- 左侧虚拟摇杆：移动
- 右侧按钮：⬆️跳跃 / ⬇️滑铲 / 🔥技能
- 武器全自动攻击

### 桌面端
- WASD：移动
- Space：跳跃
- Shift：滑铲
- 武器全自动

## 移动机制（MegaBonk风格）

| 动作 | 按键 | 效果 |
|------|------|------|
| 跳跃 | Space/⬆️ | 向上跳起（JUMP_FORCE=8, GRAVITY=20） |
| 滑铲 | Shift/⬇️ | 1.8倍速度持续0.6秒 |
| 兔子跳 | 落地瞬间跳 | 额外30%跳跃高度 |
| 加速曲线 | — | 移动有惯性（lerp 8/s加速，12/s减速） |

## 相机系统

- 固定角度第三人称（不随玩家旋转）
- 位置：玩家后方7单位，上方5单位
- 平滑跟随（移动时慢跟0.05，停下时快归0.1）
- 动态FOV（敌人多→拉远，Boss→更远）
- 屏幕震动（层次化：轻击高频/暴击中频/Boss低频）
- 顿帧（暴击0.03秒，Boss攻击0.05秒）

## 游戏节奏

| 时间段 | 事件 | 难度 |
|--------|------|------|
| 0-60秒 | 基础敌人（骷髅+蝙蝠） | ★☆☆☆☆ |
| 1-3分钟 | 加入幽灵/僵尸 | ★★☆☆☆ |
| 3-5分钟 | 加入弓手，Mini-Boss出现 | ★★★☆☆ |
| 5-7分钟 | 传送门出现，精英增多 | ★★★★☆ |
| 8-9分钟 | **Final Swarm**（双倍刷新，150上限） | ★★★★★ |
| 9分钟+ | Boss战（Tier1自动/Tier2-3需传送门） | BOSS |

## 难度分层

| Tier | 名称 | 敌人倍率 | 传送门 | 奖励倍率 |
|------|------|---------|--------|---------|
| 1 | 普通 | 1.0x | 无（Boss自动出现） | 1.0x |
| 2 | 困难 | 1.5x HP/1.3x伤害 | 1个 | 2.0x银币 |
| 3 | 噩梦 | 2.5x HP/1.8x伤害 | 2个（都要激活） | 3.0x银币 |
