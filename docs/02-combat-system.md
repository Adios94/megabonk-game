### Mini-Boss
- 3分钟后每2分钟出现一只
- 3倍HP，2倍伤害，1.5倍体型

### Final Swarm（最终狂潮）
- 第8-9分钟触发
- 刷新速度翻倍，最大数量150
- 敌人速度+20%
- 屏幕边缘红色脉冲警告

## Boss（阿努比斯）

3阶段：
- P1(100-60% HP)：近战横扫 + 地面砸击
- P2(60-30%)：召唤小兵 + 暗黑弹幕 + 近战
- P3(30-0%)：狂暴（速度+50%）+ 全屏AOE + 冲刺 + 暗黑雨

## 连击系统

- 2秒内连续击杀 → Combo计数递增
- XP倍率 = 1 + min(combo × 0.05, 1.0)，最高2倍
- Combo > 3 时HUD显示金色 "COMBO x5!"

## 伤害公式

```
最终伤害 = 武器基础伤害 × 玩家伤害倍率 × (暴击 ? 暴击倍率 : 1)
玩家受伤 = max(1, 敌人伤害 - 玩家护甲)
```
