# MegaBonk Three.js — 进度与成长系统

## 角色（3种）

| 角色 | HP | 速度 | 伤害 | 护甲 | 暴击率 | 起始武器 |
|------|-----|------|------|------|--------|---------|
| 超级猛男 Megachad | 100 | 5.0 | 1.2x | 0 | 8% | 大剑 |
| 罗伯托 Roberto | 150 | 4.0 | 1.0x | 3 | 5% | 飞斧 |
| 滑板骷髅 | 70 | 6.5 | 0.9x | 0 | 10% | 弹射骨头 |

## 武器槽解锁

起始2个武器槽，通过升级解锁更多：
- 等级5 → 第3槽
- 等级10 → 第4槽
- 等级20 → 第5槽
- 等级30 → 第6槽（上限）

任务系统也可奖励额外武器槽。

## 经验与升级

### 经验宝石

| 颜色 | 经验值 |
|------|--------|
| 绿色 | 1 XP |
| 蓝色 | 5 XP |
| 紫色 | 25 XP |
| 橙色 | 100 XP |

### 升级公式

```
所需XP = floor(10 × (1 + 等级 × 0.35))
```

最高40级。

### 升级选择

每次升级暂停游戏，弹出3个选项（有幸运典籍可多1个）：
- 新武器（如果有空槽）
- 已有武器升级（+1级）
- 新典籍 或 已有典籍升级

稀有度权重：普通60 / 稀有25 / 史诗12 / 传说3

## 连击系统

- 2秒内连续击杀 → Combo递增
- XP获取 × (1 + min(combo × 0.05, 1.0))
- 最高2倍XP加成

## 传送门系统

- Tier 1：无传送门，Boss在9分钟自动出现
- Tier 2：5分钟出现1个传送门，走近激活(3秒) → 召唤Boss
- Tier 3：需要激活2个传送门才召唤Boss

## 任务系统（30个）

跨局累积进度，完成后永久解锁奖励：

类型包括：
- 击杀数（100/500/1000/5000）
- 存活时间（2分钟/5分钟/9分钟）
- 达到等级（10/20/30/40）
- 武器进化
- Boss击杀
- 无伤存活

奖励：银币 / 解锁武器 / 解锁角色 / 额外武器槽

## 永久商店（8种升级）

用银币购买永久属性提升：

| 升级 | 最大等级 | 效果/级 | 费用递增 |
|------|---------|--------|---------|
| 最大HP | 10 | +10 | 50→1000 |
| 伤害 | 10 | +5% | 80→1200 |
| 速度 | 5 | +0.3 | 100→700 |
| 暴击率 | 5 | +2% | 120→900 |
| 拾取范围 | 5 | +0.5 | 60→450 |
| 护甲 | 5 | +1 | 100→750 |
| XP获取 | 5 | +10% | 80→650 |
| 起始等级 | 3 | +1级 | 500→2000 |

## 银币获取

每局结算：
- 击杀数 × 0.5
- 等级 × 5
- 通关奖励 +100
- Tier倍率加成（Tier2=2x，Tier3=3x）

## 存档系统

localStorage 持久化：
- 银币余额
- 商店升级等级
- 已完成任务
- 已解锁武器/角色
- 历史统计（总击杀/总局数/最佳时间/最高等级）
