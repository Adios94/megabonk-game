# 梆了个梆 · Pillar

> Stable build metadata for name / cover generation.

## §1. Core Fun

玩家在 3D 白盒战场中被骷髅、僵尸、法师、石像鬼等怪潮持续包围，通过移动、跳跃、滑铲和 bunny hop 保持距离，让自动武器不断清屏；每次升级都在武器、典籍、羁绊、遗物之间做构筑选择，最终找到飞碟召唤 Boss、击败 Boss、进入传送门推进到下一关。

核心快感是“边跑边梆”：敌人越聚越多，武器弹幕越滚越厚，玩家在近乎失控的怪潮中靠走位和构筑撕开缺口。短局内持续获得经验、银币、宝箱、神殿奖励和羁绊强化，让每一分钟都有新的战斗形态。

## §2. Pillars

### Pillar 1: 3D 机动生存

移动不是简单绕圈，而是包含跳跃、滑铲、攀爬、地形高度和 bunny hop 的空间操作。玩家需要利用高低差、平台、转角和冲刺窗口，在被包围前重置站位。理想体验是“怪潮压脸，但我还能靠身法钻出去”。

### Pillar 2: 自动武器构筑

武器自动攻击，玩家专注移动与升级选择。大剑、弹射骨头、旋转飞斧、手枪、闪电法杖、烈焰环、霰弹枪、射线枪、毒气弹、麻痹枪、虚空涟漪、灼地靴组成多种清场方式；典籍提供攻速、生命、幸运、吸引、暴击、击退等成长。构筑目标是让武器之间形成可读、可叠、可爆发的“梆梆梆”连锁反馈。

### Pillar 3: 羁绊与局内目标

游戏不只堆数值，还通过羁绊把武器组合变成战斗风格：铁血、奥术、零距寸劲、余烬轨迹、弹幕、碎骨震、弧光导体、毒师、猎标烙印等羁绊会在 T1/T2/T3 逐步改变玩法。局内还要寻找飞碟、召唤 Boss、开宝箱、充神殿、进传送门，给幸存者循环加入明确目标点。

### Pillar 4: 最终狂潮与超时压迫

常规生存期后期进入最终狂潮，敌人血量、伤害、速度和数量压力持续升高；如果玩家拖到超时，天气和氛围转向夜晚浓雾，怪物继续变强。玩家要在“继续贪发育”和“尽快召 Boss/进门”之间做风险选择。

## §3. Core Experience Loop

```
选择角色与难度
  → 进入 3D 关卡，边移动边自动攻击
  → 击杀怪物，拾取经验宝石、银币和消耗品
  → 升级选择武器 / 典籍 / 羁绊强化
  → 寻找飞碟、宝箱和充能神殿，获取额外奖励
  → 召唤并击败 Boss，开启 Boss 宝箱
  → 进入传送门或撑过最终狂潮
  → 结算奖励，回到局外商店/任务/角色解锁
```

## §4. Art Style Baseline

**Presentation mode**: `presentation_mode: depth`

**Total tone**: 美漫风 3D Roguelike Survivor。场景以风格化白盒/神殿/废墟为基础，强调清晰轮廓、强对比色块、夸张光效和密集弹幕。画面应该像一个可玩的玩具战场：角色、敌人、投射物、宝箱、飞碟、神殿都要一眼可读。

**Visual differentiation**: 玩家角色用鲜明轮廓和脚下运动反馈突出；敌人用亡灵、飞行、远程、精英等体型差异区分；Boss 采用机甲/大型单位压迫感。武器表现以可读弹道和范围提示为主：剑弧、弹跳骨头、旋转飞斧、闪电链、火圈、霰弹扇面、激光线、毒云、虚空环波、灼地痕迹都应具备明确形状。

**Pillar → Art Presentation mapping**:

| Pillar / key experience | Corresponding art presentation |
|---|---|
| 3D 机动生存 | 跳跃/滑铲拖影、地形高低差、角色落地与 bunny hop 速度感 |
| 自动武器构筑 | 多武器同时开火、弹幕层次清晰、升级后规模和频率明显增强 |
| 羁绊与局内目标 | 羁绊激活弹窗、飞碟/传送门指南针、宝箱和神殿高亮提示 |
| 最终狂潮与超时压迫 | 最终狂潮标题、夜晚浓雾、暗黑漫画调色、怪潮密度提升 |

## §5. Key Modules

### `mobility-survival`
- **Carries**: 3D 机动生存。
- **Watch-out**: 地形高度、跳跃、滑铲和 bunny hop 要服务于逃生和拉扯，不能让玩家卡住或失去方向感。

### `auto-weapon-build`
- **Carries**: 自动武器构筑。
- **Watch-out**: 武器越多越强时仍要读得清，不能让特效盖住敌人、经验宝石和交互点。

### `bond-combo`
- **Carries**: 羁绊与局内目标。
- **Watch-out**: 羁绊激活条件要让玩家能理解“为什么这几把武器能组合”，T2/T3 机制变化要比单纯加数值更有存在感。

### `boss-portal-run`
- **Carries**: 飞碟召 Boss、Boss 宝箱、传送门进关。
- **Watch-out**: 飞碟、传送门和 Boss 出现节奏必须清楚，玩家要知道下一步是找目标、召唤、击败还是撤离。
